# Phase 2: 提示工程

> **学习目标**：掌握提示工程的核心技术
> **预计时长**：1-2 周

## 📚 本阶段内容

### 1. 提示基础

- `01-prompt-anatomy.py` - 提示词结构解析
- `02-system-prompts.py` - 系统提示词设计
- `03-instruction-tuning.py` - 指令优化技巧

### 2. 进阶技术

- `04-few-shot-learning.py` - 少样本学习
- `05-chain-of-thought.py` - 思维链提示
- `06-self-consistency.py` - 自洽性提示

### 3. 结构化输出

- `07-json-output.py` - JSON 格式输出
- `08-structured-extraction.py` - 结构化信息提取

### 4. 提示优化

- `09-prompt-templates.py` - 提示词模板设计
- `10-adversarial-prompting.py` - 对抗性提示与防护

## 🎯 实战项目

**智能客服提示词系统**：设计一套完整的客服场景提示词，包括意图识别、多轮对话、情感处理等。

- [ ] 了解提示注入攻击与防护
