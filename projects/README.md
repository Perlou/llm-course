## Phase 1: 多模型对比聊天应用

> **目录**: `projects/multi-model-chat/`

### 项目描述

构建一个支持多个 LLM 的聊天应用，可以同时向多个模型发送问题并对比响应。

### 功能需求

- [ ] 支持 OpenAI、Claude、Gemini 三个模型
- [ ] 并行调用多个模型
- [ ] 流式显示所有模型的响应
- [ ] 响应时间和 Token 消耗统计
- [ ] 简洁的命令行界面

### 技术要点

- 异步并发调用
- 流式响应处理
- 错误处理与重试

### 预期成果

```
请输入问题: 什么是大语言模型？

[GPT-4] (1.2s)
大语言模型是一种基于深度学习的自然语言处理模型...

[Claude] (1.5s)
大语言模型（LLM）是使用海量文本数据训练的神经网络...

[Gemini] (1.3s)
大语言模型是能够理解和生成人类语言的 AI 系统...
```

## 项目开发建议

### 渐进式开发

1. **最小可行版本 (MVP)**: 先实现核心功能
2. **迭代优化**: 根据测试反馈改进
3. **功能扩展**: 逐步添加高级功能

### 代码规范

- 使用类型注解
- 编写单元测试
- 保持代码文档
- 遵循 PEP 8

### 版本控制

- 每个项目使用独立分支开发
- 定期提交有意义的 commit
- 使用 tag 标记里程碑
