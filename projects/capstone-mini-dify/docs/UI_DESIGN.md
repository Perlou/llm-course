# Mini-Dify - 前端页面设计文档

> 版本: v1.0  
> 更新日期: 2026-03-06  
> 技术: React 18 + TypeScript + TailwindCSS

---

## 1. 页面总览

| 页面         | 路由           | 优先级 | 说明              |
| ------------ | -------------- | ------ | ----------------- |
| 模型管理     | /models        | P0     | 供应商和模型配置  |
| Prompt 工坊  | /prompts       | P0     | Prompt 模板管理   |
| Prompt 编辑  | /prompts/:id   | P0     | 编辑/测试 Prompt  |
| 知识库列表   | /datasets      | P0     | 知识库管理        |
| 知识库详情   | /datasets/:id  | P0     | 文档管理/检索测试 |
| Agent 列表   | /agents        | P0     | Agent 管理        |
| Agent 编辑   | /agents/:id    | P0     | Agent 配置/测试   |
| 工作流列表   | /workflows     | P0     | 工作流管理        |
| 工作流编辑器 | /workflows/:id | P0     | React Flow 画布   |
| 应用列表     | /apps          | P0     | 应用管理          |
| 应用详情     | /apps/:id      | P0     | 应用配置/API Key  |
| 监控面板     | /analytics     | P1     | 数据统计图表      |

## 2. 全局布局

```
┌─────────────────────────────────────────────────────────────────┐
│  🤖 Mini-Dify                                           ⚙ 设置  │
├────────┬────────────────────────────────────────────────────────┤
│        │                                                        │
│ 模型   │                                                        │
│ Prompt │                    主内容区域                           │
│ 知识库 │                                                        │
│ Agent  │                                                        │
│ 工作流 │                  (路由对应页面)                         │
│ 应用   │                                                        │
│        │                                                        │
│ ────── │                                                        │
│ 监控   │                                                        │
│        │                                                        │
└────────┴────────────────────────────────────────────────────────┘
```

**设计要点**:

- 左侧固定导航栏（240px 宽），深色背景
- 右侧为主内容区域，浅色背景
- 顶部 Logo + 全局设置
- 导航项含图标 + 文字

## 4. 设计规范

### 4.1 品牌色

| 用途      | 颜色           | Tailwind      |
| --------- | -------------- | ------------- |
| 主色      | #6366F1 (紫蓝) | `indigo-500`  |
| 主色深    | #4F46E5        | `indigo-600`  |
| 成功      | #10B981 (绿)   | `emerald-500` |
| 警告      | #F59E0B (黄)   | `amber-500`   |
| 危险      | #EF4444 (红)   | `red-500`     |
| 背景-侧栏 | #1E1E2E (深色) | 自定义        |
| 背景-主区 | #F8FAFC        | `slate-50`    |
| 文字-主   | #1E293B        | `slate-800`   |
| 文字-次   | #64748B        | `slate-500`   |

### 4.2 排版

| 元素     | 字号        | 字重   |
| -------- | ----------- | ------ |
| 页面标题 | 24px        | Bold   |
| 卡片标题 | 16px        | Medium |
| 正文     | 14px        | Normal |
| 辅助文字 | 12px        | Normal |
| 代码     | 13px (Mono) | Normal |

使用 Google Fonts: **Inter** (正文) + **JetBrains Mono** (代码)

### 4.3 核心组件

| 组件        | 说明           | 使用场景         |
| ----------- | -------------- | ---------------- |
| Sidebar     | 左侧导航栏     | 全局布局         |
| Card        | 内容卡片       | 列表项展示       |
| Modal       | 弹窗对话框     | 创建/编辑表单    |
| CodeEditor  | 代码编辑器     | Prompt/Code 编辑 |
| ChatWindow  | 对话窗口       | Playground/预览  |
| NodePanel   | 工作流节点面板 | 工作流编辑器     |
| ConfigPanel | 配置面板       | 节点/Agent 配置  |
| StatsCard   | 统计数据卡     | 监控面板         |
| Table       | 数据表格       | 日志/列表        |
| Badge       | 状态标签       | 状态展示         |

### 4.4 响应式断点

| 断点    | 宽度       | 布局调整                  |
| ------- | ---------- | ------------------------- |
| mobile  | < 768px    | 侧栏折叠为 hamburger 菜单 |
| tablet  | 768-1024px | 侧栏收窄为图标            |
| desktop | > 1024px   | 完整侧栏 + 内容区         |

## 5. 交互设计要点

### 5.1 工作流编辑器交互

- **拖拽节点**: 从左侧面板拖拽节点到画布
- **连线**: 从节点的输出锚点拖拽到另一节点的输入锚点
- **选中节点**: 点击节点，右侧显示配置面板
- **删除**: 选中后按 Delete 键或右键菜单
- **缩放**: 鼠标滚轮缩放画布
- **运行**: 点击运行后，节点按执行顺序高亮闪烁

### 5.2 Playground 交互

- 流式打字机效果展示 AI 回复
- Agent 思考过程可折叠展示（思考→工具调用→结果）
- 对话记录可清除
- 支持 Markdown 渲染

### 5.3 文档上传交互

- 支持拖拽上传
- 上传后显示处理进度条
- 处理完成后显示切片数量统计
