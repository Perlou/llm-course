# 糖尿病

## 概述

糖尿病是一种以高血糖为特征的代谢性疾病，由胰岛素分泌不足或作用障碍引起。长期高血糖会导致多种组织器官的损害。

据国际糖尿病联盟（IDF）统计，中国糖尿病患病人数已超过1.4亿，居全球第一。

## 糖尿病分型

### 1型糖尿病

- 约占糖尿病的5%-10%
- 多在儿童和青少年发病
- 胰岛β细胞被自身免疫破坏
- 需终身依赖胰岛素治疗

### 2型糖尿病

- 占糖尿病的90%以上
- 多见于成年人，但发病年龄有年轻化趋势
- 特点是胰岛素抵抗和/或胰岛素分泌相对不足
- 与生活方式密切相关

### 妊娠期糖尿病

- 妊娠期间首次发生或发现的糖代谢异常
- 分娩后多数可恢复正常
- 未来发展为2型糖尿病的风险增加

### 其他特殊类型糖尿病

- 基因缺陷、药物或化学物质诱发等

## 危险因素

- 超重或肥胖（BMI≥24）
- 年龄≥40岁
- 糖尿病家族史
- 高血压、血脂异常
- 缺乏体力活动
- 妊娠期糖尿病史
- 多囊卵巢综合征

## 症状

### 典型症状（"三多一少"）

1. **多饮**：口渴，饮水量增加
2. **多尿**：尿量增多，夜尿次数增加
3. **多食**：食欲增加，但仍感饥饿
4. **体重减轻**：体重下降，乏力

### 其他症状

- 视力模糊
- 皮肤瘙痒
- 伤口愈合慢
- 手脚麻木、刺痛
- 反复感染

**注意**：很多2型糖尿病患者早期无明显症状，需定期检查血糖。

## 诊断标准

根据《中国2型糖尿病防治指南》：

| 诊断标准                 | 静脉血浆葡萄糖值       |
| ------------------------ | ---------------------- |
| 空腹血糖受损（IFG）      | 6.1-7.0 mmol/L         |
| 糖耐量减低（IGT）        | 餐后2h 7.8-11.1 mmol/L |
| **糖尿病诊断**           |                        |
| 空腹血糖                 | ≥7.0 mmol/L            |
| 或 餐后2h血糖            | ≥11.1 mmol/L           |
| 或 随机血糖              | ≥11.1 mmol/L（有症状） |
| 或 糖化血红蛋白（HbA1c） | ≥6.5%                  |

## 并发症

### 急性并发症

- 糖尿病酮症酸中毒（多见于1型）
- 高渗高血糖综合征（多见于2型老年患者）
- 低血糖（常见于用药不当）

### 慢性并发症

1. **大血管病变**
   - 冠心病、心肌梗死
   - 脑卒中
   - 外周动脉疾病

2. **微血管病变**
   - 糖尿病视网膜病变（可致失明）
   - 糖尿病肾病（可致肾衰竭）

3. **神经病变**
   - 周围神经病变（手脚麻木、疼痛）
   - 自主神经病变（胃轻瘫、体位性低血压）

4. **糖尿病足**
   - 足部溃疡、感染
   - 严重者需截肢

## 治疗

### 治疗目标

| 指标         | 目标值              |
| ------------ | ------------------- |
| 空腹血糖     | 4.4-7.0 mmol/L      |
| 餐后2h血糖   | <10.0 mmol/L        |
| 糖化血红蛋白 | <7.0%（个体化调整） |
| 血压         | <130/80 mmHg        |
| LDL-C        | <2.6 mmol/L         |

### 综合管理（"五驾马车"）

1. **糖尿病教育**：了解疾病知识，提高自我管理能力
2. **饮食控制**：控制总热量，均衡营养
3. **运动治疗**：规律有氧运动，每周≥150分钟
4. **血糖监测**：自我监测血糖，了解血糖变化规律
5. **药物治疗**：口服降糖药或胰岛素

### 常用药物

**口服降糖药**：

- 二甲双胍（首选）
- 磺脲类（格列本脲、格列美脲）
- 格列奈类（瑞格列奈）
- α-糖苷酶抑制剂（阿卡波糖）
- DPP-4抑制剂（西格列汀）
- SGLT-2抑制剂（达格列净）
- GLP-1受体激动剂（利拉鲁肽）

**胰岛素**：

- 1型糖尿病必须使用
- 2型糖尿病在口服药控制不佳时使用

## 日常管理

1. **定期监测**：血糖、血压、血脂、肾功能
2. **足部护理**：每日检查双足，保持清洁干燥
3. **眼底检查**：每年至少一次
4. **疫苗接种**：流感疫苗、肺炎疫苗
5. **随身携带**：糖果或葡萄糖，防止低血糖

## 预防

1. 保持健康体重
2. 均衡饮食，减少糖和精制碳水化合物摄入
3. 规律运动
4. 戒烟限酒
5. 高危人群定期筛查
