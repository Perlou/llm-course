# 胃炎与消化性溃疡

## 概述

胃炎是指各种原因引起的胃黏膜炎症，是常见的消化系统疾病。消化性溃疡主要指发生在胃和十二指肠的溃疡，与胃炎有密切关系。

## 急性胃炎

### 病因

- **感染**：幽门螺杆菌、病毒、细菌
- **药物**：非甾体抗炎药（NSAIDs）、阿司匹林
- **应激**：严重创伤、烧伤、手术
- **刺激性食物**：酒精、辛辣食物
- **其他**：胆汁反流、放射损伤

### 症状

- 上腹痛、腹胀
- 恶心、呕吐
- 食欲不振
- 嗳气、反酸
- 严重者可有呕血、黑便

## 慢性胃炎

### 分型

1. **幽门螺杆菌相关性胃炎**：最常见
2. **自身免疫性胃炎**：常伴贫血
3. **特殊类型胃炎**：化学性、放射性等

### 症状

慢性胃炎症状多不典型：

- 上腹不适、隐痛
- 饭后饱胀感
- 消化不良
- 嗳气、反酸
- 部分患者无明显症状

## 消化性溃疡

### 病因

- **幽门螺杆菌感染**：90%十二指肠溃疡、70%胃溃疡
- **药物**：NSAIDs、糖皮质激素
- **生活因素**：吸烟、饮酒、精神紧张

### 胃溃疡 vs 十二指肠溃疡

| 特征     | 胃溃疡            | 十二指肠溃疡   |
| -------- | ----------------- | -------------- |
| 疼痛时间 | 餐后半小时到1小时 | 空腹痛，夜间痛 |
| 进食影响 | 进食后加重        | 进食后缓解     |
| 好发年龄 | 中老年            | 青壮年         |
| 恶变风险 | 有                | 极少           |

### 并发症

- 出血（最常见）
- 穿孔（急症）
- 幽门梗阻
- 癌变（胃溃疡）

## 诊断

1. **胃镜检查**：金标准，可直接观察并活检
2. **幽门螺杆菌检测**：
   - 快速尿素酶试验
   - C13/C14呼气试验
   - 粪便抗原检测
3. **上消化道钡餐**：不如胃镜直观
4. **血清学检查**：胃泌素、胃蛋白酶原

## 治疗

### 根除幽门螺杆菌

标准四联疗法（14天）：

- 质子泵抑制剂（如奥美拉唑）
- 铋剂
- 两种抗生素（如阿莫西林+克拉霉素）

### 抑酸治疗

- **质子泵抑制剂（PPI）**：奥美拉唑、雷贝拉唑、埃索美拉唑
- **H2受体拮抗剂**：雷尼替丁、法莫替丁
- **抗酸药**：铝碳酸镁

### 胃黏膜保护剂

- 枸橼酸铋钾
- 硫糖铝
- 米索前列醇

## 日常调理

### 饮食建议

- 规律进食，少量多餐
- 细嚼慢咽，避免过饱
- 避免辛辣、酸性、油腻食物
- 限制咖啡、浓茶
- 戒烟限酒

### 生活方式

- 保持情绪稳定，减少压力
- 规律作息，避免熬夜
- 避免长期服用NSAIDs
- 已感染Hp者尽早治疗

## 何时就医

出现以下情况应及时就医：

- 反复上腹痛
- 呕血或黑便
- 短期内体重明显下降
- 贫血症状
- 进食困难或梗阻感
