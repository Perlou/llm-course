# 感冒与流感

## 概述

感冒和流感是最常见的呼吸道感染性疾病，虽然症状相似，但病因、严重程度和治疗方法有所不同。

## 普通感冒

### 病因

- 由多种病毒引起，最常见的是鼻病毒
- 其他病原体包括冠状病毒、呼吸道合胞病毒等
- 全年均可发病，秋冬季节多发

### 症状

- 鼻塞、流涕（初期清涕，后期可变黄稠）
- 打喷嚏
- 咽喉痛
- 咳嗽（通常较轻）
- 低热或无发热（一般≤38.5℃）
- 轻度头痛、乏力

### 病程

- 一般5-7天自愈
- 症状第2-3天最重，之后逐渐缓解

### 治疗

以对症治疗为主：

- **发热/疼痛**：对乙酰氨基酚、布洛芬
- **鼻塞**：盐水鼻腔冲洗、减充血剂（如伪麻黄碱）
- **咳嗽**：止咳药（如右美沙芬）
- **多饮水、休息**

**注意**：抗生素对病毒性感冒无效，不应滥用。

## 流行性感冒（流感）

### 病因

- 由流感病毒引起
- 分为甲型、乙型、丙型（甲型最常见，变异快）
- 主要通过飞沫传播
- 每年冬春季流行

### 症状

- **起病急**，高热（39-40℃）
- 剧烈头痛、肌肉酸痛
- 全身乏力、极度疲劳
- 呼吸道症状可能不明显
- 儿童可能有呕吐、腹泻

### 高危人群

- 65岁以上老年人
- 5岁以下儿童（特别是2岁以下）
- 孕妇
- 慢性病患者（心肺疾病、糖尿病等）
- 免疫功能低下者

### 并发症

- 肺炎（病毒性或继发细菌性）
- 心肌炎
- 脑炎
- 原有慢性病加重

### 治疗

- **抗病毒药物**：奥司他韦（达菲）、扎那米韦
  - 发病48小时内使用效果最佳
  - 高危人群发病后应尽早使用
- **对症治疗**：退热、止咳、补液
- **充分休息**

## 感冒与流感的区别

| 特征     | 普通感冒 | 流感                 |
| -------- | -------- | -------------------- |
| 起病     | 逐渐     | 突然                 |
| 发热     | 无或低热 | 高热（39-40℃）       |
| 头痛     | 轻度     | 剧烈                 |
| 肌肉痛   | 轻微     | 明显、全身酸痛       |
| 疲劳     | 轻度     | 极度疲劳             |
| 鼻塞流涕 | 明显     | 可能较轻             |
| 病程     | 5-7天    | 1-2周或更长          |
| 并发症   | 少见     | 可致肺炎等严重并发症 |

## 预防措施

### 通用预防

1. **勤洗手**：用肥皂和流水洗手至少20秒
2. **戴口罩**：在人群密集场所
3. **避免接触**：与患者保持距离
4. **通风**：保持室内空气流通
5. **增强免疫**：均衡饮食、适量运动、充足睡眠

### 流感疫苗

- **推荐人群**：6月龄以上所有人群，特别是高危人群
- **接种时间**：每年秋季（9-10月）
- **有效期**：约6-8个月，需每年接种
- **效果**：降低感染风险和重症发生率

## 就医指征

出现以下情况应及时就医：

- 高热超过3天不退
- 呼吸困难、胸闷
- 剧烈头痛或意识改变
- 严重呕吐或腹泻导致脱水
- 症状好转后再次加重
- 高危人群出现流感症状
