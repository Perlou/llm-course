# 冠心病

## 概述

冠状动脉粥样硬化性心脏病（简称冠心病），是指冠状动脉发生粥样硬化引起管腔狭窄或闭塞，导致心肌缺血缺氧或坏死而引起的心脏病。

冠心病是全球范围内导致死亡的主要原因之一，也是我国城乡居民的首位死因。

## 病因与危险因素

### 不可改变的因素

- **年龄**：男性≥45岁，女性≥55岁
- **性别**：男性发病率高于女性
- **家族史**：直系亲属有早发心血管病史

### 可改变的因素

- **高血压**：血压控制不良增加风险
- **血脂异常**：LDL-C 升高是主要危险因素
- **糖尿病**：糖尿病患者风险增加2-4倍
- **吸烟**：损伤血管内皮，促进动脉粥样硬化
- **肥胖**：特别是腹型肥胖
- **缺乏运动**：久坐生活方式

## 分型

1. **稳定型心绞痛**：劳力诱发，休息或含服硝酸甘油可缓解
2. **不稳定型心绞痛**：胸痛发作频繁或持续时间延长
3. **急性心肌梗死**：冠脉完全闭塞导致心肌坏死
4. **缺血性心肌病**：长期心肌缺血导致心脏扩大
5. **猝死**：突发心脏骤停

## 症状

### 典型心绞痛

- **部位**：胸骨后或心前区
- **性质**：压迫感、紧缩感、烧灼感
- **放射**：可放射至左肩、左臂、颈部、下颌
- **持续时间**：3-5分钟，一般不超过15分钟
- **诱因**：体力劳动、情绪激动、寒冷、饱餐
- **缓解**：休息或含服硝酸甘油

### 急性心肌梗死的警示症状

- 胸痛持续超过20分钟不缓解
- 含服硝酸甘油无效
- 伴大汗、恶心、呕吐
- 面色苍白、烦躁不安
- 有濒死感

**🚨 出现以上症状立即拨打120！**

## 诊断

1. **心电图**：ST-T 改变、Q波
2. **心肌标志物**：肌钙蛋白、CK-MB
3. **冠状动脉造影**：金标准
4. **冠脉CT血管成像**：无创检查
5. **负荷试验**：运动心电图、核素灌注显像

## 治疗

### 药物治疗

- **抗血小板药**：阿司匹林、氯吡格雷
- **降脂药**：他汀类（阿托伐他汀、瑞舒伐他汀）
- **β受体阻滞剂**：美托洛尔、比索洛尔
- **ACE抑制剂/ARB**：改善预后
- **硝酸酯类**：缓解心绞痛

### 介入治疗

- **经皮冠状动脉介入术（PCI）**：球囊扩张、支架植入

### 外科手术

- **冠状动脉旁路移植术（CABG）**：搭桥手术

## 预防与日常管理

### 一级预防

- 戒烟限酒
- 健康饮食（低盐低脂）
- 规律运动
- 控制血压、血糖、血脂
- 保持正常体重

### 二级预防（已确诊者）

- 坚持服药，不擅自停药
- 定期复查
- 避免诱发因素
- 随身携带硝酸甘油
- 学习心肺复苏
