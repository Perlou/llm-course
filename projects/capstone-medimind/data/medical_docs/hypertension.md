# 高血压

## 概述

高血压是一种常见的慢性心血管疾病，指在未使用降压药物的情况下，非同日3次测量血压，收缩压≥140mmHg和/或舒张压≥90mmHg。

根据中国高血压防治指南（2024年），我国成人高血压患病率约为27.5%，是心脑血管疾病最重要的危险因素之一。

## 病因

### 原发性高血压

约占高血压患者的90%-95%，病因尚未完全明确，可能与以下因素有关：

1. **遗传因素**：家族中有高血压病史者，患病风险增加
2. **环境因素**：
   - 高钠低钾饮食
   - 超重和肥胖
   - 长期精神紧张
   - 缺乏体力活动
   - 过量饮酒
3. **年龄因素**：随年龄增长，患病率增加

### 继发性高血压

约占5%-10%，由其他疾病引起：

- 肾脏疾病（如慢性肾炎、肾动脉狭窄）
- 内分泌疾病（如原发性醛固酮增多症、嗜铬细胞瘤）
- 睡眠呼吸暂停综合征
- 药物因素（如激素、避孕药、非甾体抗炎药）

## 症状

高血压早期多无明显症状，常见症状包括：

- 头晕、头痛（多在清晨或情绪激动时）
- 颈项僵硬
- 心悸、胸闷
- 疲劳、乏力
- 视物模糊
- 耳鸣

**注意**：高血压被称为"沉默的杀手"，很多患者没有明显症状，需定期测量血压。

## 诊断标准

根据《中国高血压防治指南》：

| 分级      | 收缩压 (mmHg) | 舒张压 (mmHg) |
| --------- | ------------- | ------------- |
| 正常血压  | <120          | 且 <80        |
| 正常高值  | 120-139       | 和/或 80-89   |
| 1级高血压 | 140-159       | 和/或 90-99   |
| 2级高血压 | 160-179       | 和/或 100-109 |
| 3级高血压 | ≥180          | 和/或 ≥110    |

## 并发症

长期高血压可导致多器官损害：

1. **心脏**：左心室肥厚、冠心病、心力衰竭
2. **脑**：脑卒中（中风）、短暂性脑缺血发作
3. **肾脏**：高血压肾病、肾功能不全
4. **眼睛**：高血压视网膜病变
5. **血管**：动脉粥样硬化、主动脉夹层

## 治疗原则

### 非药物治疗（生活方式干预）

适用于所有高血压患者：

1. **减少钠盐摄入**：每日食盐<6g
2. **控制体重**：BMI控制在18.5-24.9
3. **戒烟限酒**：白酒<50ml/天，葡萄酒<100ml/天
4. **适量运动**：中等强度有氧运动，每周5-7次，每次30分钟
5. **保持心理平衡**：减轻精神压力，避免情绪激动
6. **合理膳食**：多吃蔬菜水果，减少饱和脂肪

### 药物治疗

常用降压药物包括：

- **钙通道阻滞剂（CCB）**：如氨氯地平、硝苯地平
- **血管紧张素转换酶抑制剂（ACEI）**：如依那普利、贝那普利
- **血管紧张素II受体拮抗剂（ARB）**：如氯沙坦、缬沙坦
- **利尿剂**：如氢氯噻嗪、吲达帕胺
- **β受体阻滞剂**：如美托洛尔、比索洛尔

**注意**：药物治疗需在医生指导下进行，不可自行停药或换药。

## 日常管理

1. **定期测量血压**：建议每日固定时间测量并记录
2. **按时服药**：不要漏服、自行停药或调整剂量
3. **定期随访**：每1-3个月复诊一次
4. **认识危险信号**：出现剧烈头痛、呕吐、视物模糊等症状需立即就医

## 预防措施

1. 保持健康的生活方式
2. 定期体检，监测血压
3. 控制体重在正常范围
4. 减少精神压力
5. 积极治疗可能导致高血压的其他疾病
