## 系统简介

**MediMind 健康助手** 是一款基于 AI 的智能健康信息服务平台，旨在为您提供便捷的健康咨询服务。

### 核心特点

- 🏥 **健康问答**: 基于权威医学知识库的智能问答
- 💊 **药品查询**: 药品信息检索与相互作用查询
- 📋 **报告解读**: AI 辅助解读体检报告
- 🩺 **智能导诊**: 基于症状的科室推荐

> ⚠️ **重要提示**: 本系统仅提供健康科普信息，不能替代专业医生的诊断和治疗建议。如有紧急情况，请立即拨打 120。

## 功能介绍

### 1. 健康问答

输入您的健康问题，AI 助手会基于权威医学知识为您提供解答。

**支持的问题类型**:

- 疾病科普（症状、病因、预防）
- 健康生活建议
- 一般性健康咨询

**特色功能**:

- 多轮对话，上下文理解
- 来源引用，信息可追溯
- 紧急情况自动提醒

### 2. 药品查询

快速搜索药品信息，了解用法用量、不良反应及禁忌。

**支持功能**:

- 药品名称搜索
- 药品详情查看
- 药物相互作用检查

### 3. 报告解读

上传体检报告图片，AI 自动识别并解读检验指标。

**支持的报告类型**:

- 血常规
- 肝功能
- 肾功能
- 血脂检查

**解读内容**:

- 异常指标高亮
- 通俗易懂的解释
- 健康建议

### 4. 智能导诊

描述您的症状，AI 助手会进行多轮问诊，推荐就医科室。

**使用流程**:

1. 描述主要症状
2. 回答 AI 的追问
3. 获取科室推荐

## 使用教程

### 快速开始

1. 打开 MediMind 首页
2. 选择需要的功能模块
3. 按照页面提示操作

### 健康问答使用

```
1. 点击 "健康问答" 进入对话页面
2. 在输入框输入您的问题
3. 点击发送按钮
4. 查看 AI 回复和来源引用
```

### 药品查询使用

```
1. 点击 "药品查询" 进入搜索页面
2. 输入药品名称进行搜索
3. 点击药品卡片查看详情
4. 勾选多个药品可检查相互作用
```

### 报告解读使用

```
1. 点击 "报告解读" 进入上传页面
2. 拍照或选择报告图片上传
3. 等待 AI 分析完成
4. 查看异常指标和解读建议
```

### 智能导诊使用

```
1. 点击 "智能导诊" 开始问诊
2. 描述您的主要不适
3. 回答 AI 的问题（如持续时间、程度等）
4. 查看推荐的就医科室
```

## 常见问题

### Q1: 系统回答准确吗？

A: 系统基于权威医学知识库，信息来源可靠。但 AI 回答仅供参考，具体诊断和治疗请遵医嘱。

### Q2: 我的隐私安全吗？

A: 系统不收集、不存储您的敏感个人信息。对话记录仅用于改善服务质量。

### Q3: 报告解读支持哪些格式？

A: 目前支持 JPG、PNG 格式的清晰图片。建议拍摄时保持图片清晰、光线充足。

### Q4: 为什么药物相互作用检查提示有风险？

A: 这是基于已知药学研究的提醒，具体用药方案请咨询医生或药师。

### Q5: 导诊推荐的科室准确吗？

A: AI 导诊基于常见疾病-症状映射，可作为就医参考。如症状复杂或不确定，建议先挂全科或急诊。

## 注意事项

### ⚠️ 安全提醒

1. **本系统不能替代医生诊断**
2. 如遇以下紧急情况，请立即拨打 **120**:
   - 胸痛、呼吸困难
   - 意识不清、抽搐
   - 大量出血
   - 严重过敏反应
   - 自伤或自杀念头

### 📋 使用建议

- 描述症状时尽量详细（发作时间、程度、伴随症状）
- 上传报告图片时确保清晰可读
- 药品查询结果仅供参考，用药请遵医嘱
- 定期体检，重视异常指标
