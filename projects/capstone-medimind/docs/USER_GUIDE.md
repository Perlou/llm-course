## 系统简介

**MediMind 健康助手** 是一款基于 AI 的智能健康信息服务平台，为您提供便捷的健康咨询服务。

### 核心功能

| 功能        | 说明                         |
| ----------- | ---------------------------- |
| 🏥 健康问答 | 基于权威医学知识库的智能问答 |
| 💊 药品查询 | 药品信息检索与相互作用查询   |
| 📋 报告解读 | AI 辅助解读体检报告          |
| 🩺 智能导诊 | 基于症状的科室推荐           |
| 🗺️ 附近医院 | 定位搜索周边医疗机构         |
| ⏰ 提醒管理 | 用药/复查/测量提醒           |
| 👤 健康档案 | 个人健康信息管理             |

> ⚠️ **重要提示**: 本系统仅提供健康科普信息，不能替代专业医生的诊断和治疗建议。如有紧急情况，请立即拨打 120。

## 功能介绍

### 1. 健康问答 🏥

输入您的健康问题，AI 助手会基于权威医学知识为您提供解答。

**支持的问题类型**:

- 疾病科普（症状、病因、预防）
- 健康生活建议
- 一般性健康咨询

**特色功能**:

- 多轮对话，上下文理解
- 来源引用，信息可追溯
- 紧急情况自动提醒

### 2. 药品查询 💊

快速搜索药品信息，了解用法用量、不良反应及禁忌。

**功能特点**:

- 药品名称/功效搜索
- 药品详情查看（成分、用法、禁忌）
- 药物相互作用检查

### 3. 报告解读 📋

上传体检报告图片，AI 自动识别并解读检验指标。

**支持的报告类型**:

- 血常规、尿常规
- 肝功能、肾功能
- 血脂检查、血糖检查

**解读内容**:

- 异常指标高亮显示
- 通俗易懂的解释
- 健康建议

### 4. 智能导诊 🩺

描述您的症状，AI 助手会进行多轮问诊，推荐就医科室。

**使用流程**:

1. 描述主要症状
2. 回答 AI 的追问（持续时间、程度等）
3. 获取科室推荐和就医建议

### 5. 附近医院 🗺️

基于您的位置，搜索周边医院和诊所。

**功能特点**:

- 自动获取当前位置
- 按距离筛选 (1km/3km/5km)
- 显示医院距离和导航

### 6. 提醒管理 ⏰

设置用药、测量、复查提醒，帮助管理慢性病。

**提醒类型**:

- 💊 用药提醒
- 📊 测量提醒（血压、血糖等）
- 🏥 复查提醒
- ⏰ 其他提醒

**重复选项**:

- 单次、每天、每周、每月

### 7. 健康档案 👤

记录和管理您的健康信息。

**包含内容**:

- 基本信息（生日、身高、体重、血型）
- 过敏史
- 既往病史
- 健康记录（血压、血糖等测量值）

## 使用教程

### 快速开始

1. 访问 http://localhost:5173
2. 注册账号或登录
3. 选择需要的功能模块
4. 按照页面提示操作

### 健康问答

1. 点击首页 **"健康问答"** 卡片
2. 在输入框输入您的问题
3. 点击发送按钮或按 Enter
4. 查看 AI 回复

### 药品查询

1. 点击首页 **"药品查询"** 卡片
2. 输入药品名称搜索
3. 点击药品卡片查看详情
4. 勾选多个药品检查相互作用

### 报告解读

1. 点击首页 **"报告解读"** 卡片
2. 上传体检报告图片
3. 等待 AI 分析完成
4. 查看异常指标和解读建议

### 智能导诊

1. 点击首页 **"智能导诊"** 卡片
2. 描述您的主要不适
3. 回答 AI 的问题
4. 查看推荐的就医科室

### 附近医院

1. 点击首页 **"附近医院"** 卡片
2. 允许浏览器获取位置
3. 选择搜索距离
4. 点击医院卡片导航

### 提醒管理

1. 点击首页 **"提醒管理"** 卡片
2. 点击 **"添加提醒"**
3. 设置提醒类型、时间、重复方式
4. 允许浏览器通知权限

## 常见问题

### Q1: 系统回答准确吗？

系统基于权威医学知识库，信息来源可靠。但 AI 回答仅供参考，具体诊断和治疗请遵医嘱。

### Q2: 我的隐私安全吗？

系统使用 JWT 认证保护您的账号安全。健康数据存储在本地服务器，不会上传到第三方。

### Q3: 报告解读支持哪些格式？

目前支持 JPG、PNG 格式的清晰图片。建议拍摄时保持图片清晰、光线充足。

### Q4: 附近医院功能不显示结果？

请确保：

1. 已允许浏览器获取位置权限
2. 系统已配置高德地图 API Key

### Q5: 提醒通知没有弹出？

请检查：

1. 浏览器是否允许通知权限
2. 系统通知是否开启
3. 浏览器是否在后台运行

## 注意事项

### ⚠️ 安全提醒

1. **本系统不能替代医生诊断**
2. 如遇以下紧急情况，请立即拨打 **120**:
   - 胸痛、呼吸困难
   - 意识不清、抽搐
   - 大量出血
   - 严重过敏反应
   - 自伤或自杀念头

### 📋 使用建议

- 描述症状时尽量详细（发作时间、程度、伴随症状）
- 上传报告图片时确保清晰可读
- 药品查询结果仅供参考，用药请遵医嘱
- 定期体检，重视异常指标
- 设置提醒帮助管理慢性病
