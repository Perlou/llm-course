# MediMind 项目总结报告

> 基于 RAG + Agent + 多模态的智能健康助手平台

---

## 📋 项目概述

MediMind 是一款面向普通用户的智能健康信息服务平台，通过 AI 技术提供健康科普、药品查询、报告解读和智能导诊服务。

---

## ✨ 技术亮点

### 1. RAG 知识问答

- 基于向量检索的医学知识问答
- 来源引用，信息可追溯
- 流式响应，用户体验流畅

### 2. 安全护栏

- 紧急情况自动检测
- 输入/输出双重过滤
- 医疗合规免责声明

### 3. Agent 导诊

- 多轮对话收集症状
- 智能科室推荐
- 紧急等级评估

### 4. 多模态分析

- Gemini Vision 报告识别
- 异常指标智能解读
- 健康建议生成

### 5. 现代前端架构

- Turborepo Monorepo
- React 18 + TypeScript
- 组件化 UI 库
