## 3. 移动端适配

### 3.1 响应式断点

| 断点    | 宽度       | 布局 |
| ------- | ---------- | ---- |
| mobile  | < 640px    | 单列 |
| tablet  | 640-1024px | 双列 |
| desktop | > 1024px   | 多列 |
